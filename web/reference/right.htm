WELCOME TO YOUR SITE!

This is default.html page.

This server was setup to accept index.htm, index.html, index.asp, index.stm, default.htm, default.asp, default.stm and default.html as a default documents.

These pages contain:

  • forms to administer your web site
  • links to various info pages, FAQ's, tutorials and references
  • test pages of default server components

Webmaster could replace this page with own custom home page in the future or just call first page index.htm

All examples are located in folders /reference and /cgi-bin.
For your convenience we set up following folders:

  • /CGI-BIN
 to run executable files
  • /DATABASE
 for database files (no read permissions for web guests)
  • /DOWNLOAD
 for internet guests to download files or programs off your website
  • /SECURE
SSL enabled
  • /PROTECTED
folder protected with FTP login and password
